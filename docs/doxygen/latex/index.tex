\chapter{Ray\+Lib\+Ode\+Mech Framework }
\hypertarget{index}{}\label{index}\index{RayLibOdeMech Framework@{RayLibOdeMech Framework}}
\hypertarget{index_overview}{}\doxysection{\texorpdfstring{Overview}{Overview}}\label{index_overview}
Ray\+Lib\+Ode\+Mech is a physics framework that combines Ray\+Lib for graphics rendering with ODE (Open Dynamics Engine) for physics simulation. It provides an easy-\/to-\/use API for creating 3D physics simulations with various primitive shapes and complex composite objects.

The main functionality is documented here \doxylink{raylibODE_8c}{raylib\+ODE.\+c}\hypertarget{index_preperation}{}\doxysection{\texorpdfstring{Compiling ODE for the framework}{Compiling ODE for the framework}}\label{index_preperation}
ODE is linked in statically for convienience.

get ODE from \href{https://bitbucket.org/odedevs/ode/downloads/}{\texttt{https\+://bitbucket.\+org/odedevs/ode/downloads/}}

extract ode 0.\+16.\+6 into a directory at the same level as this project (see below)

ln -\/s ode-\/0.\+16.\+6 ode~\newline
 cd ode

I\textquotesingle{}d suggest building it with this configuration~\newline
 ./configure --enable-\/ou --enable-\/libccd --with-\/box-\/cylinder=libccd --with-\/drawstuff=none --disable-\/demos --with-\/libccd=internal

and run make, you should then be set to compile this project

Your file structure should look like this

ode~\newline
 raylib~\newline
 Ray\+Lib\+Ode\+Mech\hypertarget{index_features}{}\doxysection{\texorpdfstring{Features}{Features}}\label{index_features}

\begin{DoxyItemize}
\item Physics body creation (boxes, spheres, cylinders, capsules)
\item Composite shapes (dumbbell example shape)
\item Static trimesh support for arbitrary geometry
\item Ray picking for mouse interaction
\item Automatic collision detection and response
\item Integrated rendering with custom shaders
