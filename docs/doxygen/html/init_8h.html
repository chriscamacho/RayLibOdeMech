GraphicsContext * CreateGraphics ( int width,
int height,
const char * title )

Initialize graphics context and window.

Sets up the RayLib window, loads all models and textures, compiles shaders, and creates lighting. This must be called before any rendering can occur.

Parameters
widthWindow width in pixels
heightWindow height in pixels
titleWindow title string
Note
Enables VSync and 4x MSAA by default
Hides and locks cursor for FPS-style camera control
Loads default textures from data/ directory
