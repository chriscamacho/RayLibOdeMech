Detailed Description

Ragdoll physics simulation implementation.

This file provides a complete ragdoll (humanoid) physics simulation using ODE's joint system. The ragdoll consists of multiple body parts connected by hinge and universal joints, allowing for realistic human-like movement and posing.

Author
Chris Camacho (codifies - http://bedroomcoders.co.uk/)
Date
2026

Ragdoll Structure

Body parts (10 total):

  • Head (sphere)
  • Torso (box)
  • Left/Right Upper Arm (capsule)
  • Left/Right Lower Arm (capsule)
  • Left/Right Upper Leg (capsule)
  • Left/Right Lower Leg (capsule)

Joints (9 total):

  • Neck (hinge)
  • Left/Right Shoulder (universal)
  • Left/Right Elbow (hinge)
  • Left/Right Hip (universal)
  • Left/Right Knee (hinge)

Motor Control

The ragdoll supports motor control on all joints, making it suitable for neural network or reinforcement learning control. Use UpdateRagdollMotors() to apply forces to joints.

Create a ragdoll physics body.

Creates a complete humanoid ragdoll with articulated joints. The ragdoll is fully physics-simulated and can be used for character physics, death animations, or as a basis for reinforcement learning control.

float * motorForces )

Update ragdoll joint motors.

Applies motor forces to the ragdoll's joints for control. This is useful for neural network or reinforcement learning control of the ragdoll.

Parameters
ragdollPointer to the ragdoll structure
motorForcesArray of motor force values (one per joint)
Note
Array should have ragdoll->jointCount elements for hinge joints
For universal joints, uses ragdoll->jointCount * 2 elements
Force of 0 disables the motor for that joint
