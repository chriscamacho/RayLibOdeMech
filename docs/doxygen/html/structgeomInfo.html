Detailed Description

Stores geometry metadata for collision, texturing, and trimesh data.

  • This structure encapsulates everything needed to render and handle physics for a specific piece of geometry, including UV scaling and raw mesh indices.
Examples
arm.c, derby.c, fountain.c, and rotor.c.
